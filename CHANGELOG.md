# Changelog

本プロジェクトの注目すべき変更を記録します。形式は [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に、バージョニングは [Semantic Versioning](https://semver.org/lang/ja/) に従います。

## [1.0.0] - 2026-06-12

初の正式公開リリースです。ポモドーロ型の作業タイマーと、窓ガラスを流れる雨のアンビエント表現を組み合わせた macOS / Windows 向けデスクトップ常駐アプリ。

### 追加

#### コア体験
- 設定可能な作業/休憩サイクル（デフォルト 20分/5分）。時間管理は Rust 側の状態機械（`crates/core`）が駆動し、単体テストで遷移を固定
- 4フェーズの体験フロー: 作業（隅の HUD バーのみ）→ 予兆（休憩30秒前から透過の雨が徐々に強まる・作業継続可）→ 通り雨（雨が画面を覆う休憩・Skip/Esc でいつでも復帰）→ 雨上がり（雨がフェードアウトして作業へ）
- **セット数設定と終端フェーズ「セット終了」**: セット数は「作業の回数」を表し、最終セットの**作業が終わった時点**でタイマーが停止する（例: セット数1なら1回目の作業が終わったら終了。最終セットは休憩を挟まない）。最終セットの作業中は休憩予兆を出さない
- **セット終了画面**: セット終了時に全画面の終了画面を表示。雨上がりの虹を背景に「もう一度（セッションをやり直す）／終了（アプリを終了）」を選べる

#### 雨の表現
- raindrop-fx（WebGL2）によるリアルな雨ガラス描画（粒の合体・蛇行滑落・トレイル）
- **雨ガラスのライブ画面化（モードB）**: 実画面のキャプチャ（降雨中 2.5 秒間隔で更新）を屈折元に使用。キャプチャ不可時は静止画のモードA に自動フォールバック。オーバーレイ/HUD はキャプチャから除外されるため、スクリーンショットに雨は写らない（仕様）
- WebGL2 なし / `prefers-reduced-motion` 有効時は雨なし＋静的ベール表示に統一
- 雨音を Web Audio で自前合成（擬似ピンクノイズ＋ローパス）。通り雨でフェードイン、雨上がりでフェードアウト

#### HUD / 操作
- 隅の HUD バー（数字なしの連続バー・goal-gradient で終盤は暖色へ）。ドラッグ移動・縁をつかんで縦横斜めにリサイズ可、位置とサイズは記憶。クリックしてもフォーカスを奪わない
- HUD の直接操作（クリックで一時停止/再開など）
- **通り雨（休憩）中に残り時間（m:ss）を画面中央に表示**。雨ガラス越しに滲む控えめな表示で、クリックは奪わない
- **通り雨中のすりガラス表現を強化**: 背景のぼかしを一段深め、水滴の屈折・陰影・ハイライトを立てて「ガラスに付いた雫」を際立たせる（予兆中は背景が読める控えめなぼかしのまま）
- グローバル Esc（予兆の中止・休憩の切り上げ。作業中は登録解除して他アプリの Esc を奪わない）
- トレイ常駐: ツールチップにフェーズと残り時間（mm:ss）、一時停止/再開・Skip・設定・終了・アップデート確認

#### プラットフォーム
- **macOS 全画面 Space 共存**: 全画面アプリの上にも予兆・通り雨を表示するネイティブ実装（`NSWindowCollectionBehaviorFullScreenAuxiliary` + `canJoinAllSpaces`、フォーカス非奪取・キャプチャ除外）
- Windows 10 (1803+) / 11 対応（WebView2。未導入環境はインストーラが自動導入）

#### 設定 / 配布
- 設定UIと永続化（作業/休憩時間・セット数・音量/ミュート・起動時の自動開始）
- 自動アップデート（`tauri-plugin-updater`。起動時チェック＋トレイの「アップデートを確認…」、minisign 署名で更新の真正性を検証）
- GitHub Actions によるリリース自動化（macOS Apple Silicon/Intel・Windows のインストーラ生成、アセット名の整備、`latest.json` 配信）

### 既知の制限
- 未署名配布のため、初回起動時に OS の警告回避が必要（手順は README のインストール節を参照）
- macOS の全画面共存・クリックスルー切替は実機での合否確認が未了（実装済み・検証手順は `docs/macos-fullscreen-gate.md`）
- 背景静止画（モードA のフォールバック）はプレースホルダ
